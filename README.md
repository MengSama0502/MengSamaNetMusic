🎵 MengSama Net Music
A Next-Generation In-Game Music Experience for Minecraft

📖 中文文档
🚀 项目概览
MengSama Net Music 是一款专为 Minecraft 打造的全功能音乐播放模组，将现实世界的音乐流媒体服务无缝融入方块世界。
模组深度集成 网易云音乐 与 QQ 音乐 两大主流平台，同时支持本地音乐文件播放，并以高精度的 3D 模型与流畅的交互体验，重新定义游戏内的听觉享受。

✨ 核心特性
模块	功能亮点
🎛️ 唱片机方块	精致 3D 建模的沉浸式音乐播放器，放置于世界中，自带动态动画与光影效果
🎧 随身播放器	轻量化手持设备，右键即用；亦可放置地面作为独立音源，灵活切换
📋 智能播放列表	支持 顺序播放、随机洗牌、单曲循环 三种模式，随心掌控节奏
☁️ 网易云音乐	使用公开API接口实现无需登录账号即可听所有歌曲包含VIP
🐧 QQ 音乐	便捷搜索与播放，覆盖海量曲库
📝 实时歌词 HUD	动态滚动歌词、封面艺术、进度条同屏显示，沉浸式听歌体验
💾 智能缓存系统	自动缓存已播歌曲至本地，优化流量与加载速度，离线亦可回放
🔗 模组联动	原生支持 精妙背包 (Sophisticated Backpacks) 与 车万女仆 (Touhou Little Maid) ，扩展玩法维度
🧪 合成配方
所有配方均遵循原版风格，平衡且易于上手。

📘 使用指南
制作与部署 — 合成 唱片机方块，放置于您心仪的位置。

搜索音乐 — 通过 GUI 内嵌的搜索栏，从网易云或 QQ 音乐检索歌曲。

创建唱片/歌单 — 将搜索结果保存为 音乐唱片 或自定义 播放列表。

播放操作 — 将唱片插入唱片机或随身播放器，即刻开始播放。

便携模式 — 随身播放器支持 手持右键 调出界面，亦可 放置地面 作为小型音箱使用。

🇬🇧 English Documentation
🚀 Overview
MengSama Net Music is a high-fidelity music player mod for Minecraft, seamlessly integrating real-world streaming services into your blocky world. With deep support for Netease Cloud Music and QQ Music, plus local file playback, it delivers a premium listening experience through meticulously crafted 3D models and intuitive controls.

✨ Key Features
Module	Highlights
🎛️ Jukebox Block	Exquisitely modeled 3D music player with dynamic animations and ambient effects, placeable in-world
🎧 Portable Player	Handheld device for on-the-go listening; can also be placed as a standalone speaker block
📋 Smart Playlists	Three modes: Sequential, Shuffle, and Repeat One — full control over your queue
☁️ Netease Cloud Music	Login, search, and VIP track playback with full API integration
🐧 QQ Music	Powerful search and playback capabilities, covering massive music libraries
📝 Real-time Lyrics HUD	Synchronized scrolling lyrics, album art, and progress bar displayed in a clean overlay
💾 Intelligent Caching	Locally caches played tracks to minimize redundant downloads and enable offline replay
🔗 Mod Interoperability	Native hooks for Sophisticated Backpacks and Touhou Little Maid for expanded gameplay synergy
🧪 Crafting Recipes
Balanced recipes that respect vanilla aesthetics and progression.

📘 Usage Guide
Craft & Deploy — Craft the Jukebox Block and place it in your desired location.

Search Music — Use the built-in search bar in the GUI to find songs from Netease or QQ.

Create CDs/Playlists — Save search results as Music CDs or custom playlists.

Playback — Insert a CD into the Jukebox or Portable Player to start listening.

Portable Mode — Right-click the Portable Player in hand to open its GUI, or place it on the ground as a mini speaker.
